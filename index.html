About me

As a developer, I have experience building projects using HTML5, CSS3, Vanilla JavaScript, JQuery and React. Making my websites accessible and writing clean code are my top priorities. You can see for yourself at www.ermekcodes.com . I think you might find the “Find similar songs” project interesting. I have built it using React, Last. FM API and Firebase as my database. It pulls the data of similar songs to the user’s input from API, displays it, and stores it as a playlist in the database. I’m particularly proud of having implemented autocomplete functionality for the search bar. At the moment I’m learning Node.JS and can’t wait to update this project with the new skills.
